EmailJS Setup Guide for Your Portfolio

Follow these steps to complete your EmailJS setup and make your contact form send emails directly to your inbox.

Note: You already have your Service ID (service_3knzzcv) and Public Key (Fw-AMNBW3NIc8soDo) configured. You only need to create an email template and update the template ID in your code.

Step 1: Create an Email Template

  1. Go to https://dashboard.emailjs.com/admin/templates in your EmailJS dashboard
  2. Click "Create New Template"
  3. Design your email template. Here's a simple example:
  4. Set the "To email" as {{to_email}} or your email address directly
  5. Set the subject as "New Contact from {{user_name}}"
  6. For the content, use something like: 
    Name: {{user_name}}
Email: {{user_email}}
Message: {{message}}
  7. Save the template
  8. Important: Copy the Template ID (it looks like "template_xxxxxxx")

Step 2: Update Your Code

  1. Open your contact page file (app/contact/page.jsx)
  2. Find this line: const templateId = "template_xxxxxxx" // Replace with your template ID
  3. Replace template_xxxxxxx with your actual Template ID that you copied in Step 1

Step 3: Test Your Form

Fill out and submit your contact form. You should receive an email at milad.b.shirvan@gmail.com with the submitted information.

Troubleshooting

  • If you get an error about the template ID, make sure you've created a template in your EmailJS dashboard and copied the correct ID
  • Check the browser console for any errors
  • Make sure your EmailJS service is connected properly
  • Make sure your template has the correct parameter names (user_name, user_email, message)
  • Check your spam folder if you don't see the emails
